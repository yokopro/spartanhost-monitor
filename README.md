# 🚀 SpartanHost 补货监控系统

这是一个专门为 SpartanHost 设计的补货监控工具。支持实时库存检查、网页展示以及邮件提醒。

## ✨ 功能特点
- **实时监控**：基于 Node.js，精准抓取补货信息。
- **一键安装**：支持 Linux 环境全自动部署。
- **订阅系统**：支持用户在网页前端输入邮箱订阅补货通知。
- **可视化界面**：简洁的监控状态展示页。

## 🛠 快速安装

在你的 VPS 终端执行以下命令：

```bash
git clone [https://github.com/yokopro/spartanhost-monitor.git](https://github.com/yokopro/spartanhost-monitor.git)
cd spartanhost-monitor
chmod +x install.sh
sudo ./install.sh
